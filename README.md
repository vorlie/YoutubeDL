# Requirements
- MacOS 11.7.10 and newer.
- Or Windows 10+

## Features
- Paste youtube link
- Select video or audio (mp4/mp3)
- Select folder - Default:
  - **Windows:** `C:/Users/~/Downloads`
  - **MacOS:** `/Users/~/Downloads`
- Video info viewer
- And of course download it
